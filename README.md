# Customer Lifetime Value (CLTV) Analysis

## Overview

This project analyzes customer transaction data to calculate Customer Lifetime Value (CLTV) and identify high-value customer segments.

## Tools Used

- SQL
- Power BI
- Microsoft Excel

## Business Goals

- Understand customer revenue contribution
- Identify repeat purchase patterns
- Segment customers by value
- Support retention planning

## Key Metrics

- Customer Lifetime Value
- Purchase Frequency
- Revenue by customer
- High-value customer segmentation

## Project Highlights

- Analyzed 3+ years of customer transaction data
- Built SQL queries for customer segmentation
- Prepared data in Excel
- Built interactive Power BI dashboard
- Identified top 15% high-value customers

- SQL
- Data Cleaning
- KPI Reporting

- Business Analysis
