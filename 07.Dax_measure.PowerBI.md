# Power BI DAX Measures — Sales Performance Dashboard

## How to Add These
In Power BI Desktop: **Home → New Measure** → paste the DAX → rename appropriately.
Organize all measures in a dedicated `_Measures` table (create an empty table with just the name, put all measures there).

---

## 📌 SECTION 1 — Base Revenue Measures

```dax
Total Revenue = SUM(sales_data[sales])
```

```dax
Total Profit = SUM(sales_data[profit])
```

```dax
Total Orders = DISTINCTCOUNT(sales_data[order_id])
```

```dax
Total Quantity = SUM(sales_data[quantity])
```

```dax
Profit Margin % =
DIVIDE(
    [Total Profit],
    [Total Revenue],
    0
)
```

```dax
Avg Order Value =
DIVIDE(
    [Total Revenue],
    [Total Orders],
    0
)
```

```dax
Avg Discount % =
AVERAGEX(
    sales_data,
    sales_data[discount]
)
```

## 📌 SECTION 2 — Time Intelligence Measures

```dax
Revenue LY =
CALCULATE(
    [Total Revenue],
    SAMEPERIODLASTYEAR('Date'[Date])
)
```

```dax
Revenue YOY Change =
[Total Revenue] - [Revenue LY]
```

```dax
Revenue YOY % =
DIVIDE(
    [Revenue YOY Change],
    [Revenue LY],
    0
)
```

```dax
Revenue MTD =
CALCULATE(
    [Total Revenue],
    DATESMTD('Date'[Date])
)
```

```dax
Revenue QTD =
CALCULATE(
    [Total Revenue],
    DATESQTD('Date'[Date])
)
```

```dax
Revenue YTD =
CALCULATE(
    [Total Revenue],
    DATESYTD('Date'[Date])
)
```

```dax
Revenue Prior Month =
CALCULATE(
    [Total Revenue],
    PREVIOUSMONTH('Date'[Date])
)
```

```dax
Revenue MOM % =
DIVIDE(
    [Total Revenue] - [Revenue Prior Month],
    [Revenue Prior Month],
    0
)
```

## 📌 SECTION 3 — Ranking Measures

```dax
Product Revenue Rank =
RANKX(
    ALLSELECTED(sales_data[product_name]),
    [Total Revenue],
    ,
    DESC,
    DENSE
)
```

```dax
Top 10 Products Filter =
IF([Product Revenue Rank] <= 10, 1, 0)
```

```dax
Rep Revenue Rank =
RANKX(
    ALL(sales_data[sales_rep_name]),
    [Total Revenue],
    ,
    DESC,
    DENSE
)
```

## 📌 SECTION 4 — Target vs Actual

```dax
Target Revenue = SUM(sales_targets[target_amount])
```

```dax
Revenue vs Target =
[Total Revenue] - [Target Revenue]
```

```dax
Target Achievement % =
DIVIDE(
    [Total Revenue],
    [Target Revenue],
    0
)
```

```dax
Target Status =
SWITCH(
    TRUE(),
    [Target Achievement %] >= 1,    "✅ On Target",
    [Target Achievement %] >= 0.85, "⚠️ Near Target",
    "❌ Below Target"
)
```

## 📌 SECTION 5 — Customer Measures

```dax
Total Customers = DISTINCTCOUNT(sales_data[customer_id])
```

```dax
New Customers =
CALCULATE(
    DISTINCTCOUNT(sales_data[customer_id]),
    FILTER(
        sales_data,
        YEAR(sales_data[order_date]) =
            YEAR(CALCULATE(MIN(sales_data[order_date]),
                ALLEXCEPT(sales_data, sales_data[customer_id])))
    )
)
```

```dax
Revenue per Customer =
DIVIDE(
    [Total Revenue],
    [Total Customers],
    0
)
```

```dax
Customer Concentration % =
-- % revenue from top 10 customers
VAR Top10Revenue =
    CALCULATE(
        [Total Revenue],
        TOPN(10, VALUES(sales_data[customer_id]), [Total Revenue], DESC)
    )
RETURN
    DIVIDE(Top10Revenue, [Total Revenue], 0)
```

## 📌 SECTION 6 — Dynamic KPI Card Titles (for card visuals)

```dax
KPI Revenue Title =
"Total Revenue  " &
FORMAT([Revenue YOY %], "+0.0%;-0.0%;0.0%") & " YOY"
```

```dax
KPI Margin Color =
-- Use in conditional formatting: green if >20%, amber if >10%, red otherwise
SWITCH(
    TRUE(),
    [Profit Margin %] >= 0.20, "Green",
    [Profit Margin %] >= 0.10, "Amber",
    "Red"
)
```

## 📌 SECTION 7 — Date Table (create this first!)

```dax
Date =
ADDCOLUMNS(
    CALENDAR(DATE(2022,1,1), DATE(2024,12,31)),
    "Year",         YEAR([Date]),
    "Quarter",      QUARTER([Date]),
    "Quarter Label", "Q" & QUARTER([Date]) & "-" & YEAR([Date]),
    "Month Number", MONTH([Date]),
    "Month Name",   FORMAT([Date], "MMMM"),
    "Month Short",  FORMAT([Date], "MMM"),
    "Week Number",  WEEKNUM([Date]),
    "Weekday",      FORMAT([Date], "dddd"),
    "Is Weekend",   IF(WEEKDAY([Date],2) >= 6, TRUE, FALSE),
    "YearMonth",    FORMAT([Date], "YYYY-MM")
)
```
**Mark this table as a Date Table:** Right-click → Mark as Date Table → Date column.

## 📌 SECTION 8 — Power BI Report Pages Setup

### Page 1 — Executive Overview
- KPI Cards: Total Revenue, Profit, Orders, Margin %
- Line chart: Monthly Revenue Trend (with YOY overlay)
- Bar chart: Revenue by Region
- Donut chart: Revenue by Category
- Table: Top 10 Customers

### Page 2 — Product Analysis
- Matrix: Category → Sub-Category Revenue + Margin
- Bar chart: Top 10 Products by Revenue
- Scatter plot: Revenue vs Profit Margin (by sub-category)
- Table: Loss-making products

### Page 3 — Sales Rep Performance
- Table: Rep Scorecard (Revenue, Orders, Margin, Target %)
- Gauge chart: Overall Target Achievement
- Bar chart: Rep Revenue Ranking
- Line chart: Rep Monthly Trend (use slicer to filter)
